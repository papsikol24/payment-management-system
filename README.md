# Online Payment Management System

A comprehensive payment management system for educational institutions built with Node.js, Express, and MySQL.

## Features

- Student payment management
- Multiple payment methods (GCash, Bank Transfer, Credit Card, Cash)
- Role-based access control
- Real-time transaction tracking
- Secure authentication system

## Installation

1. Clone the repository
2. Run `npm install`
3. Set up MySQL database
4. Configure environment variables
5. Run `npm start`

## Technologies Used

- Backend: Node.js, Express.js
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Authentication: Custom session management
